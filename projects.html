CodeXplorer: Coding Wiki for Beginners

CodeXplorer is a coding wiki especially made for beginners. This wiki teaches C#, C++, Java and Python.

Developed with Visual Studio 2019 and Xamarin, the coding wiki was done by a team of IT students, including Izenn Hart, in CvSU Silang Campus in ~2024-2025.

The app features coding information from programming languages such as C#, C++, Java and Python and games (multiple-choice, fill in the blanks, and guess the output) which was sourced from W3School for the convenience of students to an app which has an easy access to their lessons with just one installation to their smartphones.

CvSU Student Form Manager

CvSU Student Form Manager is a software that prints registration form and certificate of grades for students by teachers.

Developed with Visual Studio 2019 and XAMPP, the student form manager was done by young Izenn Hart in ~2023-2024 with group mates Mark Ulyses and Maria Angelica as a project for their course.

The student form manager features CRUD and printing of registration form and certificate of grades by admins for each semester of the students.
